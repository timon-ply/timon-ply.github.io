## 13. Drittanbieter-Werbung

Die App kann Werbeanzeigen von Google LLC über das Google Mobile Ads SDK („AdMob") anzeigen. Diese Werbeanzeigen werden auf einer Belohnungsbasis angeboten – Sie können sich freiwillig entscheiden, eine Werbeanzeige anzusehen, um dafür In-App-Credits zu erhalten. Ohne Ihre ausdrückliche Handlung werden keine Werbeanzeigen angezeigt.

Das Google-Werbe-SDK kann bestimmte Geräte- und Nutzungsdaten erheben, um Werbeanzeigen auszuliefern, darunter Gerätekennungen, IP-Adressen und App-Interaktionsdaten. Diese Daten werden gemäß der [Datenschutzerklärung von Google](https://policies.google.com/privacy) verarbeitet.

So deaktivieren Sie personalisierte Werbung:
- **iOS:** Einstellungen → Datenschutz & Sicherheit → Apple-Werbung → „Personalisierte Werbung" deaktivieren
- **Android:** Einstellungen → Google → Werbung → Personalisierung der Werbung deaktivieren

Weitere Einstellungen unter [https://adssettings.google.com](https://adssettings.google.com).
